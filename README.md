# Metodología de Análisis Estadístico: Contrastes de Hipótesis

## Descripción

Este proyecto presenta una metodología integral y organizada para el análisis de contrastes de hipótesis estadísticas, con un enfoque académico riguroso y visualizaciones interactivas.

## Contenido

La página web incluye:

### 1. Marco Metodológico
- Objetivos de la investigación
- Protocolo de análisis paso a paso
- Diagrama de flujo metodológico
- Fases de implementación

### 2. Conceptos Fundamentales
- Nivel de significación (α)
- Error Tipo I y Error Tipo II
- P-valor y su interpretación correcta
- Potencia estadística (1 - β)
- Relación entre niveles de significación
- Tabla de decisiones

### 3. Análisis Visual
Cinco gráficos interactivos generados con Chart.js:

1. **Distribución de Errores**: Visualización de errores tipo I (α) y tipo II (β) bajo H₀ y H₁
2. **Potencia vs Tamaño Muestral**: Relación entre n y la potencia para diferentes niveles α
3. **Interpretación del P-valor**: Representación gráfica del p-valor en la distribución
4. **Comparación de Niveles**: Trade-off entre error tipo I y potencia
5. **Simulación**: Distribución de p-valores bajo H₀

### 4. Ejemplos Prácticos
- Prueba de media poblacional
- Análisis de potencia estadística
- Comparación de proporciones
- Datos reales con interpretaciones

### 5. Evaluación de Afirmaciones
Análisis detallado de 7 afirmaciones comunes sobre contrastes de hipótesis, con:
- Evaluación (Verdadero/Falso)
- Justificación teórica
- Correcciones cuando aplica
